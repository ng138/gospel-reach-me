Política de Privacidad para NFCBible

Fecha de entrada en vigor: 22 de mayo de 2025

Esta Política de Privacidad explica cómo NFCBible ("nosotros", "nos" o "nuestro") recopila, utiliza y protege la información cuando visitas nuestro sitio web a través de etiquetas NFC o acceso directo. Estamos comprometidos a proteger tu privacidad y manejar tus datos de manera transparente y en cumplimiento con el Reglamento General de Protección de Datos (RGPD) y otras leyes de privacidad relevantes.

1. Qué Información Recopilamos

Recopilamos los siguientes tipos de información:

Datos de Uso Anónimos:

  • País de Origen (derivado de IP): Cuando accedes a nuestro sitio web, Cloudflare, nuestra red de distribución de contenido, nos proporciona automáticamente el código de país ISO de dos letras derivado de tu dirección IP (encabezado CF-IPCountry). Esto se utiliza únicamente para determinar tu preferencia de idioma predeterminada y para estadísticas geográficas anónimas y agregadas sobre "activaciones del evangelio". Tu dirección IP completa no es almacenada por nosotros.
  • ID de Usuario Anónimo (UUID): Para fines funcionales, nuestro sitio web genera y almacena un identificador único y anónimo (UUID v4) en el almacenamiento local de tu navegador (no una cookie tradicional). Este ID nos ayuda a recordar tus preferencias de idioma/versión y permite el seguimiento anónimo y agregado de "activaciones del evangelio" desde tu navegador/dispositivo específico. Este ID no te identifica personalmente.
  • Datos de Interacción: Rastreamos recuentos anónimos de cargas de página, actualizaciones de versículos y cambios de idioma/versión para entender la participación del sitio web. Estos recuentos se agregan globalmente y por país.

Información Proporcionada Voluntariamente:

  • Tu Nombre: Si eliges usar nuestra función "Enviar Tu Nombre", proporcionas voluntariamente tu nombre. Este nombre se asocia con tu ID de Usuario anónimo (UUID) y se utiliza únicamente con el propósito de reconocer anónimamente "con quién se compartió este evangelio" en nuestras estadísticas agregadas. No intentamos vincular este nombre con tu identidad personal.

Información de Terceros (Google AdSense):

  • Nuestro sitio web utiliza Google AdSense para mostrar anuncios. Google AdSense puede recopilar y utilizar datos sobre tus visitas a este y otros sitios web para proporcionar anuncios sobre bienes y servicios de tu interés. Esto puede implicar el uso de cookies y tecnologías similares por parte de Google y sus socios. Consulta la Política de Privacidad de Google para obtener más información sobre cómo manejan los datos.

2. Cómo Utilizamos Tu Información

Utilizamos la información recopilada para los siguientes propósitos:

  • Funcionalidad del Sitio Web: Para proporcionar las características principales de nuestro sitio web, incluyendo:
    • Configurar automáticamente tu idioma predeterminado según tu país.
    • Recordar tus preferencias de idioma y versión de la Biblia.
    • Mostrar versículos bíblicos aleatorios.
    • Habilitar la función "Enviar Tu Nombre" para reconocimiento anónimo.
  • Análisis y Estadísticas: Para entender los patrones de uso del sitio web, medir el alcance global del "evangelio" y mejorar nuestros servicios. Todos los datos analíticos son agregados y anonimizados.
  • Seguridad: Para proteger nuestro sitio web de actividades maliciosas, como ataques DDoS, utilizando los servicios de seguridad de Cloudflare, incluido Turnstile para la verificación de bots.
  • Publicidad: Para mostrar anuncios relevantes a través de Google AdSense, lo que ayuda a mantener el funcionamiento de este sitio web.

3. Base Legal para el Procesamiento (RGPD)

Procesamos tus datos en base a los siguientes fundamentos legales:

  • Intereses Legítimos: Procesamos datos de uso anónimos (país de origen, ID de usuario anónimo, datos de interacción) para operar, mantener y mejorar nuestro sitio web, y para entender su alcance global. Este procesamiento es necesario para nuestros intereses legítimos en proporcionar un servicio funcional y atractivo, y estos intereses están equilibrados con tus derechos de protección de datos.
  • Consentimiento: Cuando envías voluntariamente tu nombre, proporcionas consentimiento explícito para que procesemos esta información con el propósito declarado de reconocimiento anónimo. Puedes retirar este consentimiento en cualquier momento.
  • Necesidad Contractual: Para ciertas funcionalidades del sitio web que solicitas explícitamente (por ejemplo, interactuar con el formulario de envío de nombre), el procesamiento de tu ID anónimo es necesario para cumplir con esa solicitud.

4. Compartir y Divulgar Datos

No compartimos, vendemos ni alquilamos tus datos personales a terceros.

  • Proveedores de Servicios: Utilizamos Cloudflare como nuestra red de distribución de contenido y plataforma sin servidor. Cloudflare procesa datos en nuestro nombre para proporcionar sus servicios (por ejemplo, enrutamiento de tráfico, protección DDoS, almacenamiento KV/R2, Objetos Durables). Cloudflare está comprometido con el cumplimiento del RGPD.
  • Google AdSense: Como se mencionó, Google AdSense opera de forma independiente y puede recopilar datos con fines publicitarios. No controlamos las prácticas de datos de Google.
  • Requisitos Legales: Podemos divulgar tu información si es requerido por ley o en respuesta a solicitudes válidas de autoridades públicas (por ejemplo, una orden judicial o agencia gubernamental).

5. Retención de Datos

  • ID de Usuario Anónimo: Almacenado en el almacenamiento local de tu navegador hasta que borres los datos del navegador.
  • Nombres Enviados Voluntariamente: Almacenados en Cloudflare KV durante el tiempo necesario para nuestros propósitos estadísticos anónimos, típicamente 2 años, después de los cuales pueden ser anonimizados aún más o eliminados.
  • Estadísticas Agregadas: Los recuentos globales y específicos por país se conservan indefinidamente ya que están anonimizados y no identifican a individuos.

6. Tus Derechos de Protección de Datos (RGPD)

Bajo el RGPD, tienes los siguientes derechos con respecto a tus datos:

  • Derecho de Acceso: Tienes derecho a solicitar copias de tus datos personales que tenemos (si se recopilan datos identificables, que son mínimos en este proyecto).
  • Derecho de Rectificación: Tienes derecho a solicitar que corrijamos cualquier información que creas que es inexacta o que completemos información que creas que está incompleta.
  • Derecho de Supresión ("Derecho al Olvido"): Tienes derecho a solicitar que eliminemos tus datos personales bajo ciertas condiciones. Para el ID de Usuario anónimo, puedes hacer esto borrando el almacenamiento local de tu navegador. Para los nombres enviados voluntariamente, puedes contactarnos para solicitar la eliminación.
  • Derecho a la Limitación del Procesamiento: Tienes derecho a solicitar que restrinjamos el procesamiento de tus datos personales bajo ciertas condiciones.
  • Derecho a Oponerse al Procesamiento: Tienes derecho a oponerte a nuestro procesamiento de tus datos personales bajo ciertas condiciones.
  • Derecho a la Portabilidad de Datos: Tienes derecho a solicitar que transfiramos los datos que hemos recopilado a otra organización, o directamente a ti, bajo ciertas condiciones.
  • Derecho a Retirar el Consentimiento: Si dependemos de tu consentimiento para procesar tus datos, tienes derecho a retirar ese consentimiento en cualquier momento.

Para ejercer cualquiera de estos derechos, contáctanos utilizando los detalles proporcionados a continuación.

7. Cookies y Tecnologías Similares

  • Almacenamiento Local (ID de Usuario Anónimo): Nuestro sitio web utiliza localStorage para almacenar tu gospel_user_id anónimo. Este es un mecanismo de almacenamiento del lado del navegador, similar a una cookie, pero no se envía con cada solicitud HTTP. Ayuda a proporcionar funcionalidad consistente (como recordar tu preferencia de idioma) y contribuye a estadísticas de uso anónimas. Al continuar utilizando nuestro sitio web, consientes el almacenamiento de este ID anónimo en tu almacenamiento local.
  • Cookies de Google AdSense: Google AdSense, como servicio de publicidad de terceros, puede utilizar cookies y tecnologías similares (por ejemplo, balizas web) para recopilar información sobre tus visitas a este y otros sitios web para proporcionar anuncios dirigidos. Estas cookies son controladas por Google. Puedes administrar tu configuración de anuncios de Google o optar por no recibir publicidad personalizada visitando la Configuración de Anuncios de Google.
  • Banner de Consentimiento de Cookies: Para cumplir con el RGPD, implementamos un banner de consentimiento de cookies en tu primera visita al sitio web. Este banner te informa sobre el uso del almacenamiento local para el ID de Usuario anónimo y cookies de terceros (como Google AdSense) y requiere tu consentimiento explícito antes de que estos se activen. Tienes la opción de aceptar todos, rechazar todos (lo que puede afectar la funcionalidad como la visualización de anuncios), o administrar tus preferencias.

8. Medidas de Seguridad

Implementamos medidas técnicas y organizativas razonables para proteger tu información contra acceso no autorizado, divulgación, alteración o destrucción. Esto incluye el aprovechamiento de la robusta infraestructura de seguridad de Cloudflare, incluida la protección DDoS y la verificación de bots Turnstile.

9. Cambios a Esta Política de Privacidad

Podemos actualizar nuestra Política de Privacidad de vez en cuando. Te notificaremos de cualquier cambio publicando la nueva Política de Privacidad en esta página y actualizando la "Fecha de entrada en vigor" en la parte superior. Te recomendamos revisar esta Política de Privacidad periódicamente para cualquier cambio.
